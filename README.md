# Mon Acupunctrice Hub

Application web de gestion de contenu pour @Mon_acupunctrice — calendrier éditorial, banque d'idées, génération de captions IA, et distribution multi-plateforme.

## Stack
- React + Tailwind CSS
- Firebase Firestore
- Hébergé sur Render

## Phases de développement
- **Phase 1 (MVP) :** Calendrier éditorial + banque d'idées
- **Phase 2 :** Génération de captions IA (Claude API)
- **Phase 3 :** Distribution multi-plateforme (Meta, YouTube, TikTok, Pinterest)
- **Phase 4 :** Tendances, analytics, collaboration

## Démarrage local
```bash
npm install
npm run dev
```
